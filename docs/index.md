# splitwise

A Node.js and Express-based expense sharing application that helps users track group expenses, split bills, and manage balances between friends.

## Tech Stack

- **Language:** Node.js
- **Framework:** Express

## Running Locally

```bash
npm install
npm run dev
```

Service runs on port `3000`.

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/` | Service info |
| GET | `/health` | Health check |
